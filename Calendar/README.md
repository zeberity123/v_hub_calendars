# Calendar

## 환경

> - Frontend
>   - HTML, CSS, JS
> - Backend
>   - Django
> - DB
>   - MySQL
>
> - 실행 방법
>
> ```shell
> $ python manage.py runserver
> ```



## 기능

### main page

- today 클릭 시 현재 날짜로 이동

### 일정 추가

- 달력을 클릭하면 일정을 추가할 수 있다 
- 하루종일을 체크하면 시간입력 X
